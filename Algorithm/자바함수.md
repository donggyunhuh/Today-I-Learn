## 3.배열
```java
int arr[] = {1,6,3,5,2};

//오름차순
Arrays.sort(arr);

//내림차순
Arrays.sort(arr, Collections.reverseOrder());

//1~6 일부만 정렬
Arrays.sort(arr, 1, 6);

//오름차순 정렬 후 binary search
Arrays.binarySearch(arr,2);

//배열을 ArrayList로 변환할때!
List list = Arrays.asList(arr);

//배열의 특정 범위 자르기
int tmp[] = Arrays.copyOfRange(arr,0,3);
```
