## HashMap

```java
//1.선언
HashMap<String, Integer> hm = new HashMap<>();

//2.key-value 넣기
hm.put("java", 0);

//3.키로 값 가져오기   
hm.get("java");

//4.containsKey()로 존재유무 확인
if (!hm.containsKey("java")) hm.put("java", 1);

//5. 특정 키가 없으면 값 설정, 있으면 기존 값 가져오는 함수
hm.put("java", hm.getOrDefault("java", 3));

//6.keySet() 함수로 맵 순회
for(String key : hm.keySet()){
    hm.get(key);
}
```
